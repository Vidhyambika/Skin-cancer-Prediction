# Skin-cancer-Prediction

A mini-project developed and submitted as part of my third-year college coursework. This project demonstrates a simple web application that predicts skin diseases and cancer based on user-provided inputs. It is shared here as a learning resource for students and developers interested in healthcare AI applications.

## Features

- Predicts skin diseases and potential cancer risks using 34 input parameters.

- Accepts real-time user inputs through a web interface.

- Built as a lightweight web application using Python and Flask.

## Technologies Used

**Language:** Python

**Web Frameworks:** Flask, Werkzeug, Jinja2

**IDE/Environment:** Spyder

## Installation

1. Clone the repository:

```bash
git clone https://github.com/Vidhyambika/Skin-cancer-Prediction.git
```

2. Navigate into the project directory:

```bash
cd Skin-cancer-Prediction
```


3. Install required packages:

```bash
pip install -r requirements.txt
```


4. Run the Flask app:

```bash
python app.py```
```

## Usage

- Open your browser and go to http://127.0.0.1:5000/.

- Fill in the 34 required input parameters.

- Submit the form to get predictions for skin disease and cancer detection.
